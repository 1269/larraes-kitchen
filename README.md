# Larrae's Kitchen

Marketing site for Larrae's Kitchen, a soul food caterer in Benicia, CA.

## Requirements
- Node 22.12+ (see `.nvmrc`)
- pnpm 9 (see `packageManager` in `package.json`)

## Commands
- `pnpm dev` — Astro dev server on http://localhost:4321
- `pnpm build` — Production build
- `pnpm check:images` — Image-budget CI check (wired in a later wave)
